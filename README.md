# outlook-chrome-extension

Outlook Web の予定表で重複予定を検出する Chrome 拡張の最小サンプル。

## 使い方

1. Chrome の拡張機能管理 (`chrome://extensions/`) を開く
2. 右上の「デベロッパー モード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択
4. Outlook Web の予定表を開く
5. 右上に出る「重複検出」ボタンを押す

## 動作概要

- Outlook Web 予定表内の `[data-calitemid]` を予定要素として収集
- 週表示のカラムごとに表示位置が重なる要素を重複候補としてハイライト
- 既存属性 `data-conflict="1"` がある要素も重複扱いに追加
- 「重複をクリア」でハイライトを解除

## 注意

- Outlook の DOM は変更される可能性があります
- 日本語/英語のキャンセル・辞退予定は検出対象から除外しています
